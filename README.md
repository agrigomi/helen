# Helen is a lite and flexible HTTP server
### It supports:
* HTTP/1.1 (for now)
* SSL
* Virtual hosts
* gzip and deflate compression
* url and error handling
* JSON configurable

### Dependencies:
* libssl-dev
* libz-dev

### How to get
~~~
git clone https://github.com/agrigomi/helen.git
~~~

### How to build
~~~
cd helen
./configure
make
~~~
